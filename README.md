# SSI Grade Calculator ✦ Calculateur de notes Master SSI

Salut ! C'est ton petit calculateur de notes pour le **Master SSI – Sécurité des Systèmes Informatiques** à l'ESST 🛡️💻  
J'ai codé ça avec amour pour nous simplifier la vie pendant les partiels 😅

**Stack que j'ai kiffé utiliser :**  
React 18 + Vite ⚡ + Tailwind CSS 🎨 + Framer Motion pour les petites animations smooth  

---
### Pour lancer le projet en 2 secondes, Install dependencies (install Node.js to run npm commands to install , https://nodejs.org )

```bash 
npm install
npm run dev     # → ouvre direct http://localhost:5173 ✨

Et pour la version pro :
```bash
npm run build   # tout se met dans /dist
```

---
### Déployer en un clin d'œil sur Vercel (c'est mon truc préféré 🚀)

**Option facile (drag & drop) :**  
Fais `npm run build` → va sur https://vercel.com/new → glisse le dossier `/dist` → et voilà !  

**Option que je recommande (GitHub) :**  
1. Pousse tout sur ton repo GitHub (comme celui-là https://github.com/justcallme-tchi/ssi-calculator)  
2. Va sur vercel.com → New Project → Import Git Repository  
3. Vercel détecte tout seul que c'est du **Vite** → clique **Deploy** et c'est parti !  

Aucun paramètre bizarre à toucher, ça marche direct 💙

---
### La fameuse formule (pour pas se tromper dans les calculs 😭)

**Pour un module complet (CC1 + CC2 + TP + Examen) :**  
moyCC = (CC1 + CC2) / 2  
moyCCTP = (moyCC × 2 + TP × 3) / 5  
note finale = (moyCCTP × 2 + Exam × 3) / 5  

**Moyenne du semestre :**  
S1 → (UEF × 18 + UEM × 9 + UED × 1 + UET × 2) / 30  
S2 → (UEF × 18 + UEM × 9 + UED × 2 + UET × 1) / 30
